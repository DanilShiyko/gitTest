Основные команды Git 1-го семинара

> **git init** - создание локальнго репозитория (добавится скрытая папка Git)


> **git status**  - получить информацию от git  о его текущем состоянии

>**git add .** - добавить файл или файлы к следующему коммиту


>git commit -am "message" - добавление файлов в отслеживание и создание коммита

>git log - вывод на экран ситории всех коммитов с из хеш-кодами
git log --graph - представление логов с графами

>git branch - помторет список веток в репозитории

>git branch **<название ветки>** - создать новую веку

>git checkout <название ветки>   - переход к другой ветке

>git checkout -b <название ветки> - создание ветки и переход к ней
 
 >git branch -D  <название ветки> - удалить ветку

 >git checkout <название ветки>^ - переход на предпоследний коммит ветки которую указали, если указать ^^, то перейдет к предпоследней предпоследней ветке, в зависисмоти от того сколько укажешь ^ так и будет в глубь переходить

 
 
 ## Основные команды Git 2-го семинара
 >git log --oneline  - лог в сокращенном виде

 >git merge <название ветки которую присоединяем> - слияние веток

 >git merge --abort - отменить сление, которое прошло с конфликтом

 >git branch -m <старое название ветки> <название в которое хотим переименовать>     - изменить название ветки

 >git log --oneline  - лог в сокращенном виде
 >git log --all --oneline - показывает все логи


 ## Семинар 3: заливаем ветки на удаленный репозиторий GitHub

>git remote add origin https://github.com/DanilShiyko/gitTest.git - создает на удаленном репозитории в облаке репозитории origin
>git branch -M main - переименовывает текущую ветку в main
>git push -u origin main
>git remote set -url origin https://github.com/DanilShiyko/gitTest.git - переименовывает переменную origin той ссылкой которую назначаем (в случае ошибки: already exists)

>git clone <url-адрес репозитория> - клонирование внешнего репозитоирия на локальный ПК

> git pull  -получение изменений и слияние с локальной версией
> git push - отправляет локальную версию репозитория на внешний
